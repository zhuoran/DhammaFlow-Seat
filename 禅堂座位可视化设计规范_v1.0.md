# 禅堂座位可视化页面设计规范（Cinema-Style）

> Version 1.0 · 专业 UI/UX 设计方案 · 可直接整合进设计文档或 PRD  
> 适用于 DhammaFlowSeat 禅堂座位管理模块

---

# 🎞️ 1. 设计理念概述

禅堂座位的可视化采用类似“电影院选座系统”的视觉与交互模式，但整体保持禅修中心需要的“简洁、宁静、清晰、有序”的设计风格。

核心目标：

- 提升座位图可读性  
- 支持自动分配、手动微调  
- 清晰呈现区域、性别、冲突等信息  
- 易操作、易打印、移动端友好

# 🧩 3. 座位组件（SeatItem）设计规范

### 座位包含：

- 座位编号  
- 状态颜色  
- Hover 信息  
- 点击反馈  

### 🎨 座位颜色体系

| 状态 | 颜色 | 描述 |
|------|-------|--------|
| 空座位 | `#E0E7FF` | 柔和浅紫蓝，清晰但不突兀 |
| 男众 | `#4F6FAE` | 稳定的藏青色（低饱和） |
| 女众 | `#C7B2D6` | 柔和淡紫藤色 |
| 不可用 | `#F5F5F5` | 占位但不引人注意 |
| Hover | 白色边框 + 微阴影 | 影院式高亮 |
| Selected | 深色边框 + 稍深背景 | 用于人工选择调整 |

# 🧭 5. 区域（Zone）设计规范

每个区域有标题，如：

```

║   区域 A（男众前区）  ║
╚══════════════╝
```

区域使用 **淡灰色背景块 + 内部座位网格**。

# 🖱️ 6. 交互流程

### **① 生成布局**
用户设置：
- 行数 / 列数  
- 区域配置  
- 默认性别/座位模板  

点击 **生成布局** → 生成电影院风格的座位图。

---

### **② 点击座位 → 打开右侧抽屉（Drawer）**
包括：

```
座位信息（行/列/编号/区域）
当前学员（如有）
选择学员（搜索）
设置性别 / 标记为不可用
保存
```

类似影院系统的座位选中侧栏。

---

### **③ 自动分配预览**
- 已分配 → 座位填充颜色  
- 冲突 → 黄色高亮或带警告边框  
- 未分配 → 白色边框 + 闪烁引导（可选）  

---

### **④ 打印视图**
打印时自动生成：

- 黑白或淡灰简化布局  
- 可选显示：姓名 / 编号 / 性别  
- 支持 A4 / A3  

---

# 📱 7. 移动端适配设计

- 座位图可水平滚动（overflow-x: auto）  
- 点击座位 → 信息在底部展开，而不是 Drawer  
- 缩小座位尺寸（20~26px）  
- 顶部控制台折叠为“设置”按钮
