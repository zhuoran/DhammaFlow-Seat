
## 🧠 角色设定
你是一位 **前端可视化与打印优化专家**，擅长为基于 Web 的管理系统设计 **打印友好、结构清晰的页面布局（A4格式）**。
你的目标是优化一个“禅修中心房间与座位分配结果页面”，让它在 **网页端美观清晰，在 A4 打印时整齐、低信息密度、禅意极简风格**。

---

## 🧾 当前页面内容
- 页面显示各楼层的房间列表（第1层、第2层、第3层……）
- 每个房间卡片包括：房间号、房型（新生房/老人房）、容量、入住者姓名、年龄、床位编号
- 顶部有汇总信息：房间总数、学员人数、已入住房间数

---

## 🎯 优化目标
1. 页面结构清晰、适合 A4 打印（横向或纵向）
2. 每个楼层分区清晰、分页合理
3. 房间卡片整齐、信息层级清楚
4. 黑白打印仍清晰可读（通过灰度区分）
5. 不使用复杂框架或依赖（纯 HTML + CSS）
6. 极简、安静、空间感强，适合禅修环境展示

---

## 🧩 实现要求

### 📄 布局结构
- 顶部显示统计信息（房间总数 / 学员人数 / 已入住房间数）
- 按楼层分区展示，每层有标题条（浅灰底、加粗）
- 每层下为房间卡片网格布局（建议每行 2~3 个）
- 每层独立 section，在打印时不跨页

### 🎨 房间卡片样式
- 固定宽高，卡片大小一致（视觉整齐）
- 房号 + 房型使用更大字号、加粗显示
- 床位信息简化为符号（①② 或 🛏1 🛏2）
- 使用浅灰边框（#CCCCCC）、圆角（6px）
- 卡片内容字号 10–12pt，行距 1.3

### 🧘 打印样式优化（@media print）
- 页面边距 10–15mm
- 禁止分页中断房间卡片 (`page-break-inside: avoid;`)
- 背景色与边框在打印中保留 (`print-color-adjust: exact;`)
- 每层标题前加分页符（除第一页）
- 适配黑白打印，灰度区分区域

### ⚙️ 技术实现
- 使用原生 HTML + CSS（无 Tailwind、无 Bootstrap）
- 数据可来源于 JS 渲染或静态模板
